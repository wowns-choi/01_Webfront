레이아웃(layout)

        사전적 의미 : 배치, 정리
        기술적 의미 : 구성 요소를 제한된 공간에 효율적으로 배치하는 것을 의미한다.

화면 배치 방법(형식) : display 속성 

        요소가 화면에 어떻게 보여질지 형식을 지정하는 속성
        - block : 화면을 수직 분할(행을 나눔) + width / height 속성 사용 가능
        - inline : 화면을 수평 분할 + width / height 속성 사용 불가능
        - inline-block : inline의 수평분할 + block 크기 조정 가능
        - none : 화면에 요소가 표시되지는 않으나 존재하고 있는 상태
        - flex : 요소의 정렬되는 방향, 요소간의 간격을 유연하게 처리하는 형식

화면 (영역) 분할

상하 2분할

        준비물
        -감싸는 영역 요소가 필요하다.
        -내부의 영역을 분할할 요소가 필요하다.
        -display:block 을 사용해서 해볼 것이다.
        -크기단위 (고정(px) / 가변(%))

상하 3분할

        준비물
        -감싸는 영역 요소가 필요하다. : id="container-2"
        -화면 분할 내부 요소 : class="div-2"
        -분할 비율 -> 20 : 30 : 50
        -배경색 자유롭게

좌우 2분할

        준비물
        - 감싸는 영역 요소
        - 내부 분할 요소
        - 크기 단위
        - display : inline-block 사용해서 해볼거임(약간 문제점 있음)

display: none

        요소는 존재하지만 화면에는 보이지 않음(아예 렌더링 자체가 안됨. 즉, 레이아웃상 공간을 차지하지 않음.)

        -> visibility : hidden; (투명하지만 레이아웃상 공간은 차지함)
