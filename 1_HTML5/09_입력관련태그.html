input 태그

웹 문서에서 사용자가 입력할 수 있는 양식을 제공하는 태그

input 태그의 모든 속성은 inline 이다.

type="text"

        -한 줄 짜리 문자열(텍스트)을 입력할 수 있는 입력 상자
        -input 태그의 type 속성의 기본값은 text다.
    
        
type이 text일 때의 사용가능한 속성

        - size : 입력상자 크기
        - maxlength : 입력 받는 텍스트의 최대길이(글자수)
        - placeholder : 입력 상자에 작성될 내용을 설명

form 태그

        - div 와 같은 영역(block 형식)
        - 내부에 작성된 input 태그의 값을 서버로 전달하는 역할을 한다.
        
        [속성]
        1. action : 내부에 작성된 값을 전달할 서버 또는 페이지 주소를 작성
        2. name : form 태그의 이름을 지정하여 form 태그 구분할 때 사용할 수 있다.
        3. method : 서버로 데이터 전달하는 방식을 지정하는 속성 (아무것도 안쓰면, default 가 get요청)
        4. target : action 에 지정된 주소를 현재탭/새탭 중 어디서 열지 지정
