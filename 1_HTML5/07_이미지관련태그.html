img 태그

        -웹 문서에 사진, 그림 같은 이미지를 삽입하는 용도의 태그

        [속성]
        1) src : 삽입할 이미지의 경로를 지정하는 속성 (파일경로, 웹주소)
        2) width / height : 이미지의 크기를 지정하는 속성
        3) alt : 이미지를 설명하는 속성 => 이미지가 출력되지 않는 경우 화면에 직접 표시해준다.

src 속성 확인

width, hight 속성을 이용한 크기 조절(고정/가변 크기)

고정 크기 단위(px) : 화면을 마우스로 늘리든 말든 고정된 크기

가변 크기 단위(%) : 화면이나 영역 기준으로 사이트 크기에 따라 이미즈이 크기가 변경되는 단위

alt 속성 확인
