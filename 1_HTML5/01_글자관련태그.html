글자 관련 태그

b태그 : 단순히 글자를 굵게 표시하는 태그
i 태그 : 단순히 글자를 기울이는 태그



strong 태그
em 태그

mark 태그 : 형광펜 그어놓은 듯한 효과
u 태그 : underline 밑줄
s 태그 : 취소선(잘못됨을 표시)
del 태그 : 삭제선(없어진 텍스트)
small 태그 : 일반텍스트보다 살짝 작은 크기의 텍스트 sup 태그(윗첨자) : x2
sub 태그(아랫첨자) : log2
abbr 태그 : 마우스 오버 시 툴팁 출력
HTTTP


태그들은 해당 태그 내에서 중첩으로 사용 가능하다.
굵은글자를 사용할 수도 있고, 기울이거나 취소선을 넣는 등
다양하게 사용할 수 있다.
