비교 연산자

      - 좌변과 우변을
        같다, 같지 않다, 초과, 미만, 이상, 이하  비교해서
        맞을 경우 true, 아닐 경우 fasle를 반환하는 연산자
        (연산 결과로 boolean(논리 값)을 반환)
  
  
      - 복합 기호를 사용할 때 등호(=)는 무조건 오른쪽에 작성!!!

입력된 두 수가 같은지 확인



좌변에 입력한 수가 우변의 입력한 수보다 큰지 확인



배수가 맞는 확인



5은/는 2의 배수 : false

복합 대입 연산자(+=, -=, *=, /=, %=)

        - 기존 변수에 저장 되어있는 값에
        새롭게 대입되는 값의 산술 연산 결과를 대입


        (증감 연산자는 1씩 증가 감소,
        복합 대입 연산자 원하는 만큼 증가, 감소)


        ex)
        let temp = 10;
        
        temp += 5; //  temp = temp + 5;  (누적/차감 효과)

현재 값 : 0

논리 연산자( &&, ||, ! )

        AND 연산자(A && B)
        - A,B 모두 true일 경우 결과로 true를 반환하는 연산자
        (나머지 경우는 false)


        - 문장에서
        ~와, 그리고(이고), 이면서(면서), 부터, 까지, 사이
        라는 글귀가 있으면 AND 연산을 수행


        - ex) 104는 100 이상의 수 이면서 짝수인가?
        - ex) 50은 70 이하이고, 4의 배수인가?
        - ex) 13은 1 부터 10 사이의 수가 맞는가?


        --------------------------------------------


        OR 연산자(||)
        - A,B 모두 false일 경우 결과로 false를 반환하는 연산자
        (나머지 경우는 true)


        - 문장에서
        또는, ~거나(이거나)
        라는 글귀가 있으면 OR 연산을 수행


        - ex) 4는 10을 초과 하거나, 짝수인가?
        - ex) 50은 0 이하 또는 40 이상인가?




        --------------------------------------------


        논리 부정(NOT) 연산자(!)
        - 논리 값을 반대로 바꾸는 연산자


        ex) let test = true;
            !test; //  !true -> false

삼항 연산자 ( 조건식 ? true인 경우 : false인 경우 )

      - 조건식의 결과에 따라서 수행되는 코드가 다른 연산자
  
  
      (조건식은 항상 결과로 true 또는 false가 반환되도록 작성)
